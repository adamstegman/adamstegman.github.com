My Kind of Stupid is my personal blog.

My blog contains my thoughts mostly on development topics, but may also touch on other areas. Generally it will consist of things I encounter in my daily work.

Based on a project on Github called Nanoc::Heroku, it is a website consisting of only static files compiled by a Ruby library called Nanoc.

I write the markup primarily in Markdown but HTML is also available. I write the styles in Sass and scripts in JavaScript.
